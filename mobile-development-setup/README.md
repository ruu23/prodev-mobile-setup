# Mobile Development Setup with Expo

## Project Overview
This project is part of the **Mobile Development with React Native** track.  
The goal of this stage is to set up the mobile development environment using **Expo**, test it on a physical device using **Expo Go**, and create the first mobile application using the **Expo Router template**.

---

## Task 0: Setting Up and Testing the Mobile Development Environment

### Tools Used
- Node.js (LTS)
- Visual Studio Code (VS Code)
- Operating System: Windows
- Physical Device: Mobile phone (Android / iOS)
- Expo Go App

---

### Steps Followed

1. Confirmed that **Node.js LTS** and **VS Code** were already installed.
2. Visited the official Expo Go website: https://expo.dev/go
3. Installed **Expo Go** on my physical device from:
   - Google Play Store (Android) / App Store (iOS)
4. Opened the Expo Go app.
5. Logged in using an Expo account.

---

### Challenges Faced
- No major issues were encountered during installation.
- The setup process was smooth, and Expo Go worked correctly after login.
